DropLHa について

DropLHa はファイルやフォルダを圧縮して LHA 書庫を作成するアプリケーションです。LHA は国内の Windows PC ユーザの間でよく使われていたファイル圧縮形式です。

DropLHa で書庫を作成するもっとも簡単な方法は、圧縮したいファイルやフォルダを DropLHa のアイコンにドラッグアンドドロップすることです。ドラッグされたファイルやフォルダは圧縮されて LHA 書庫に格納されます。また、書庫のファイルはデフォルトではデスクトップに作成されます。例えば「書類」フォルダをドラッグすると、デスクトップに「書類.lzh」という名前のファイルができます。

DropLHa の使い方

DropLHa のアイコンに圧縮したいファイルやフォルダをドラッグします。

  1. Finder で書庫にしまいたいファイルやフォルダを選択します。
  2. それらを DropLHa のアイコンが反転するまでドラッグします。
  3. マウスのボタンを離します。DropLHa はそれらのアイテムを書庫にしまい、終了すると自動的に終了します。

メニューからアイテムを選択して書庫を作成するには：

  1. DropLHa アイコンをダブルクリックします。
  2. ファイルメニューから "開く..." を選びます。
  3. ファイル選択ダイアログが表示されますので書庫にしまいたいファイルを選びます。（Shift や Command キーを押しながらクリックすると複数のアイテムを選択できます。）
  4. "選択" をクリックします。

環境設定

DropLHa 環境設定では、書庫の形式や書庫へファイルを格納するときの動作を変更することができます。

圧縮タブ

圧縮方式：
LHA 書庫の圧縮方式を指定します。デフォルトは lh5 です。
注意： LHA 書庫を扱うユーティリティの中には lh6, lh7 をサポートしていないものがあります。
ヘッダレベル：
LHA 書庫のヘッダレベルを指定します。デフォルトは 2 です。
隠しファイルを含める：
このオプションが ON のとき、DropLHa は .DS_Store などの隠しファイルも書庫に含めます。
アイテムごとに別々の書庫を作成する：
このオプションが ON のとき、複数のファイルやフォルダをドラッグしたときそれらを別々の書庫ファイルにしまいます。通常は1個の書庫ファイルにしまいます。
ドラッグ＆ドロップで圧縮後、終了しない：
このオプションが ON の場合、DropLHa はドラッグアンドドロップで起動され書庫を作成した後も終了しなくなります。

出力先タブ

デスクトップ：
デフォルトでは、書庫ファイルはデスクトップにつくられます。
オリジナルと同じ：
書庫ファイルは、圧縮したファイルと同じフォルダにつくられます。
その都度指定：
圧縮しようとするたびに書庫をつくるフォルダを訊ねます。
指定：
DropLHa はここで指定したフォルダにいつも書庫ファイルをつくります。
